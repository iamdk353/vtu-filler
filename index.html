Quick Setup (3 Steps)

Follow the steps below to get running in about 3 minutes.

1

Get a Free Gemini API Key

  1. Open Google AI Studio. here
  2. Sign in with your Google account.
  3. Select Create API Key. on top right corner
  4. Choose and existing Project or create a new one
  5. Copy the key that starts with "AIza".
2

Install the Extension

  1. Download the extension folder extension folder. here
  2. Unzip the extension folder.
  3. Open Chrome side bar > Extension > Manage Extension
  4. Turn on Developer mode. on top Right
  5. Click Load unpacked and select the extracted folder
3

Configure and Test

  1. Click the extension icon in Chrome. top right puzzel like icon
  2. Click click on Ai Form Filler
  3. Paste your API key and click Save.
  4. Open your internship website. here and login
  5. Select the internship and date.
  6. open AI Form Filler , Use Generate with AI to fill the form.

Daily Usage

Once set up, the daily flow is fast and consistent.

  1. Open the form page.
  2. Type a one-line work description.
  3. Click Generate with AI.
  4. Review and click Fill Form.
  5. Click Save Data for next time.

Manual Entry

Prefer to write everything yourself? Use manual entry fields in the popup.

  • Work Summary and Learnings are required.
  • Hours Worked supports 0.25 steps.
  • Reference links are optional.
  • Skills are comma-separated values.

Troubleshooting

Common issues and quick fixes.

Invalid API key

Confirm the key starts with "AIza" and was copied fully. If needed, generate a new one.

AI takes too long

Initial generation may take up to 10 seconds. Check your connection and try again.

Form not filling

Reload the form page, then try again. If the form structure changed, update selectors in content.js.

Privacy and Security

Your data stays on your machine except for Gemini API requests.

  • API key is stored locally in Chrome storage.
  • Data is sent only to Gemini for generation.
  • No tracking, analytics, or third-party services.
