## Troubleshooting

### "Invalid API key" error?
- Make sure you copied the entire key (starts with "AIza...")
- Generate a new key from Google AI Studio
- Check if key is activated (may take 1-2 minutes)

### AI takes too long?
- Wait up to 10 seconds for first generation
- Subsequent generations are faster
- Check your internet connection

### Form not filling?
- Make sure you're on the correct form page
- Reload the page and try again
- Check console (F12) for errors
